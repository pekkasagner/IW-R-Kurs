tidyverse

Das tidyverse in R ist eine Sammlung von Paketen, denen dieselbe Repräsentation von Daten (tidy data) zu Grunde liegen und die ein gemeinsames API Design nutzen. “Tidy data” umschreibt “aufgeräumte” bzw. “saubere” Daten und damit Daten, die so aufbereitet sind, dass jede Variable in einer Spalte und jede Beobachtung in einer Reihe repäsentiert wird.

Installation

# Install from CRAN
install.packages("tidyverse")

Nutzung

Der Befehl library(tidyverse) lädt automatisch die folgenden Pakete, die sofort genutzt werden können.

  • ggplot2, für Datenvisualisierung,
  • dplyr, für Datenbearbeitung,
  • tidyr, für das Säubern von Daten,
  • readr, für den Import von Daten,
  • purrr, für funktionales Programmieren,
  • tibble, für Tibbles (eine Art DataFrame),
  • stringr, für Strings,
  • forcats, für Faktor-Variablen.

*Die oben angegeben Weblinks enthalten neben der Beschreibung der jeweiligen R-Pakete auch sogenannte “Cheatsheets”. Dort werden sehr übersichtlich die wichtigsten Funtkionen der Pakete dargestellt und erläutert.

Weitere Pakete

Neben den bisher genannten Paketen, die sofort genutzt werden können, wenn das tidyverse-Paket geladen wird, installiert der Befehl install.packages("tidyverse") weitere Pakete. Diese werden seltener gebraucht, sind aber doch so gebräuchlich, dass sie direkt mit installiert werden. Über den library("Paketname") Befehl müssen diese allerdings je nach Anwendungsbedarf zusätzlich in die Arbeitsumgebung geladen werden. Diese mit installierten Befehle umfassen:

  • Für die Arbeit mit Daten mit zeitlichem Bezug:

    • hms, für Uhrzeiten,
    • lubridate, für Datums/Zeit-Transformationen.

  • Für den Import weiterer Datenformate:

    • feather, für das Teilen von Daten mit Python und weiteren Programmiersprachen,
    • haven, für SPSS, SAS and Stata Daten
    • httr, für Web-APIs,
    • jsonlite für JSON,
    • readxl, für .xls und .xlsx Dateien,
    • rvest, für Web-Scraping,
    • xml2, für XML,

  • Modellierung

    • broom, für das Säubern / Auslesen von Modellobjekten.

ggplot2

ggplot2 stellt die Hauptbiblothek für Datenvisualisierung mit R in diesem Kurs dar. Das Paket ist ein System für die Erstellung von Visualisierungen basierend auf The Grammar of Graphics. ggplot2 erlaubt dem Nutzer die Erstellung von Datenvisualisierungen in verschiedenen, aufeinander aufbauenden Schichten (Layern) und bietet dabei höchste Flexibilität.

Installation

Wie oben beschrieben ist der einfachste Weg ggplot2 zu installieren einfach das komplette tidyverse zu installieren. Wenn gewünscht, kann das Paket aber auch einzeln installiert werden.

# Der einfachste Weg:
install.packages("tidyverse")

# Die Alternative:
install.packages("ggplot2")

ggplot2 Grammar Guide (Englisch)

Diese Online Ressource bietet einen konzisen Überblick über die wichtigsten ggplot2 “Vokabeln”.

ggplot2 grammar guide

Data Vizalisation - A Practical Introduction (Englisch)

Das Buch von Kieran Healy ist eines der besten und umfassendsten Werke über Datenvisualisierung mit ggplot2. Neben der hier frei verfügbaren Online Version befindet sich auch eine Hard Copy bei uns im Haus.

R for Data Science (Englisch)
